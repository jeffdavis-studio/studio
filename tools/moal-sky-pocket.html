The Framing Shift

The Apr 17 prep brief recommended leading with De Coded. That recommendation predates Erin and Sydney's Apr 19 email framing, which was specifically about Mez rotation. Go with the frame Erin raised. Don't argue for De Coded unless Erin or Sydney reopens it.

What "Mez rotation" likely means The Mez is their large-scale immersive gallery. "Rotation" means The Sky would cycle with other programming on some time cadence rather than installing as a dedicated solo show. This is a lower-commitment path for the museum and a lower-lift first step for Jeff.

Three Asks

Specific commitments to try to extract from Erin or Sydney during the visit. Any one of these is a win.

Ask 1 · Scale decision

Which scale of The Sky are we talking about?

Point Sydney to the three-scale spec (small room / gallery wall / immersive install). Ask which fits the Mez programming rotation they have in mind. This grounds every follow-up.

Ask 2 · Programming cadence

What does the rotation look like?

Weekly? Monthly? Seasonal? Part of a mixed evening? Jeff needs to understand how The Sky slots into what Sydney already programs, so he can design for it.

Ask 3 · Technical pilot

Can we do a short pilot run in the Mez?

Frame it as a low-stakes technical test. Render The Sky output at Mez spec, run it for a defined window, see how it sits in the space. This de-risks the decision for MoA+L and gives Jeff production data.
